tdc测试数据存储
===
**Http api 测试工具**

###功能

----------


1. 显示项目列表，修改项目名称(双击修改)，删除项目
2. 显示下面模块列表(单击项目显示模块)，修改模块名称(双击修改)
3. 测试用例可添加，保存，修改
4. 测试用例添加，删除，测试和分析按钮
5. 测试用例添加用例说明
6. 数据库兼容access，h2(默认)
7. 模块可拖动到其他项目
8. 添加只读页面
9. http头可添加参数
